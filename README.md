# 🍴 Spoon Restaurant Website

Welcome to the **Spoon Restaurant Website**, a sleek and modern web interface designed to showcase a restaurant's offerings in an elegant, user-friendly way. This project was built using **HTML**, **CSS**, and **JavaScript**, with a focus on responsive design, animations, and a clean UI/UX experience.

🔗 **Live Demo**: [Visit Site](https://shubham-cyber-prog.github.io/Spoon-Website/)  
📂 **Repository**: [GitHub Repo](https://github.com/Shubham-cyber-prog/Spoon-Website.git)

---

## 🖥️ Features

- Responsive design for all screen sizes
- Clean and modern user interface
- Scroll and hover animations
- Dedicated sections for:
  - Hero / Landing
  - About Us
  - Menu
  - Chef’s Section
  - Awards / Recognition
  - Gallery
  - Contact Us / Newsletter

---

## 📌 Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Styling and animations
- **JavaScript** – DOM interactions and interactivity
